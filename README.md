# Game Title: Alien Catcher

## Game Description:
Alien Catcher is a fast-paced browser game where players catch falling aliens using a spaceship. The goal is to score as many points as possible before time runs out, while avoiding bombs that reduce your score.

## How to Play:
- Click "Start" to begin.
- Use Arrow Left/Right keys to move the spaceship.
- Catch cyan aliens to earn +10 points.
- Avoid red bombs or lose -15 points.
- The game ends when the timer reaches zero.

## Key Features and Concepts Used:
- HTML for structure and semantic layout
- CSS for styling, responsiveness, and visual feedback
- JavaScript for game logic:
  - Variables, arrays, functions, loops, and conditionals
  - DOM manipulation and event handling
  - Continuous game loop (60 FPS) until win/loss
  - Distinct states: Start → Playing → Game Over
  - Visible scoring and timer
  - Replay/Restart option
  - Visual feedback for player actions
